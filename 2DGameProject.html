2D Platformer Game

A Unity project featuring sprite animations, movement scripts, and camera controls.

About This Project

This Unity 2D platformer game was developed as a demonstration of game design principles. It includes features like animated sprites for characters and objects, scripts for player movement, and dynamic camera tracking to enhance the gameplay experience.

Key Features

  • Sprite animations for walking, jumping, and idle states
  • Player movement script for smooth controls
  • Camera script to follow the player dynamically
  • Engaging level design with obstacles and platforms
