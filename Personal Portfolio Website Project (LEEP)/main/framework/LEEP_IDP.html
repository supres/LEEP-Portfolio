LEEP Identity Quest Project

Volunteer Experience: Exploring Identity through True Colors Assessment

About This Project

The LEEP Identity Quest Project was part of a volunteer experience where LEEP Interns explored their true identities through the True Colors Assessment, a tool designed to figure out personal strengths and preferences in work and life.

Technologies Used

  • Canva: Used for creating presentation materials
  • True Colors Assessment: A tool for identifying personal traits and preferences
